Struktur i JavaScript
==================================
[FIGURE src=image/webapp/backpack.jpg?w=c5 class="right"]

Vi har nu bra struktur på vår CSS/SASS kod och tiden har nu kommit för att ta ett steg i rätt riktning även för JavaScript koden. Sista delen av kursmoment 1 var att dela upp koden för Me-appen i olika filer för att få en bättre struktur på koden. Vi ska i denna övning titta på verktyg för att strukturera vår JavaScript kod. Målet är att vi bara importerar en fil i `index.html` och att vi använder modulerna på ett bättre sätt än vi har gjort tidigare.
